# AdobeXD-Swap-Positions
A Adobe XD plugins that swaps the position of two selected nodes.

## Usage

1. Select two nodes.
2. Select `Plugins->Swap Position` or use keyboard shortcut `Cmd+Shift+X`(Mac) or `Ctrl+Shift+X`(Windows)
